iCareTechCR - Sistema de Gestión 🚀


Bienvenido al repositorio oficial de iCareTechCR, una plataforma diseñada para la gestión eficiente de servicios tecnológicos, reparaciones y sistemas de seguridad.

🛠️ Tecnologías Utilizadas

Este proyecto está construido con un stack moderno y eficiente:

•	Backend: Python con Flask.
•	Base de Datos: SQLite para un almacenamiento ligero y rápido.
•	Frontend: HTML5, CSS3 y JavaScript (utilizando componentes de Bootstrap para un diseño responsivo).
•	Control de Versiones: Git & GitHub.

📋 Funcionalidades Principales
•	Gestión de Inventario: Control detallado de repuestos y dispositivos.
•	Registro de Servicios: Seguimiento de reparaciones técnicas y mantenimientos de cámaras de seguridad.
•	Interfaz Adaptable: Visualización óptima tanto en desktop como en dispositivos móviles.

🚀 Instalación y Uso
	1.	Clona este repositorio:
    
git clone https://github.com/TravisRivera14/iCareTechCR.git

	2.	Instala las dependencias necesarias:
pip install flask

	3.	Ejecuta la aplicación:
python app.py
